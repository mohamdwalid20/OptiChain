# 🚀 OptiChain

OptiChain is a full-featured **Inventory & Supply Chain Management System** built using ASP.NET Core MVC.  
The system is designed to help organizations efficiently manage products, suppliers, and orders with a clean and user-friendly interface.

---

## 📌 Features

### 📦 Inventory Management
- Add new products
- Edit existing products
- Delete products
- View product status (In stock / Low stock / Out of stock)
- Search, filter, and pagination

### 🏢 Supplier Management
- Add / Edit / Delete suppliers
- Supplier details management
- Consistent UI with Products module

### 🧾 Orders System
- Create and manage orders
- Order items tracking
- Order status monitoring
- Dynamic statistics for orders

### 👨‍💼 User System
- Authentication (Login / Register)
- Role-based access (Managers / Workers)
- Google Sign-In integration (if implemented)
- Forgot password with email verification
